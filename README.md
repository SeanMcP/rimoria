# Rimoria

⌨️ A text-based adventure game in Python

[Project Board](https://github.com/SeanMcP/rimoria/projects/1)

## How to play

1. Clone the repository

    ```sh
    git clone https://github.com/SeanMcP/rimoria.git
    ```
2. Go into directory

    ```sh
    cd rimoria
    ```
3. Run `index.py` in Python 3.5+

    ```sh
    python index.py
    ```
